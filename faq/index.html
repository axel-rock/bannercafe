FAQ

What is called a "creative" on Bannercafe?

A creative represents a single banner. For now, it has to be an HTML5 banner. In the future, more formats might be supported.

What is called a "campaign" on Bannercafe?

A campaign is a collection of creatives. You can see it as a collection, or a folder.

My creative has the wrong dimensions

Include the dimensions in the file name for better results: "[width]x[height]"
